API Migration System v1.0 - Installation and Usage Guide
========================================================

This is a standalone executable version of the Autonomous API Version Migration System.

System Requirements:
- Linux (64-bit)
- Python 3.8+ (already bundled in the executable)
- GUI desktop environment (for the graphical interface)

Installation:
1. Download and extract the API_Migration_System_v1.0.zip file
2. Make the executable runnable: chmod +x API_Migration_System
3. Double-click the executable or run it from terminal: ./API_Migration_System

Usage:
1. Launch the application
2. Select source directory (containing code to migrate)
3. Select target directory (where migrated code will be saved)
4. Click "Run Migration" to start the process
5. Monitor progress in the application window
6. Use "Export Report" to save detailed migration reports
7. Use "Rollback" if you need to undo changes

Features:
- Automatic API migration with formal verification
- Proof certificates for transformation correctness
- GUI interface for easy operation
- Rollback capabilities
- Detailed progress reporting

For more information, see README.md and DEMO_README.md

Note: This executable is self-contained and includes all necessary dependencies.
